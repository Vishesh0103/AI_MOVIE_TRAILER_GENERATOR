# AI Movie Trailer Generator

A beginner-friendly AI project that generates creative movie trailer scripts based on user prompts.  
Built using Python in a Jupyter Notebook environment.

## Features
- Generate unique movie trailer ideas instantly
- Simple prompt-based interaction
- Easy to understand and modify

## Tech Stack
- Python
- AI / LLM (e.g., Gemini API)

## How to Use
1. Open the `.ipynb` notebook
2. Run all cells
3. Enter your movie idea or prompt
4. Get a generated trailer script

## Notes
1. Make sure your API key
2. You can customize prompts for better results
